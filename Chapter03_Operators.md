# 第 3 章：运算符（Operators）

   1. 赋值运算符：`=`
   2. 算术运算符：`+`、`-`、`*`、`/` 和 `%`
   3. 比较运算符：`>`、`>=`、`<`、`<=`、`==` 和 `!=`
   4. 逻辑运算符：`&&`、`||`、`!` 和 `^`
   5. 位运算符：`&`、`|`、`~`、`^`、`>>`、`<<`
   6. 复合赋值运算符：`+=`、`-=`、`*=`、`/=`、`%=`、`&=`、`|=`、`^=`、`>>=` 和 `<<=`
   7. 自增自减运算符：`++`、`--`（前缀、后缀）
   8. 条件运算符：`?:`
